# Smart City: Dashboard Prediksi Kualitas Udara Real-Time

Aplikasi Flask berbasis Machine Learning (RandomForest) untuk memprediksi kadar polusi udara PM2.5 secara real-time. Proyek ini mengintegrasikan pengumpulan data dari Open-Meteo, pelatihan model dengan Scikit-Learn dan MLflow, serta deployment otomatis menggunakan Docker ke Hugging Face Spaces.
